# プライバシーポリシー

UtaMemo（以下、「本アプリ」）は、ユーザーのプライバシーを尊重し、以下の方針に基づいて情報を取り扱います。

## 収集する情報
本アプリでは、アプリの品質向上および不具合解析のため、以下の情報を収集する場合があります。

- アプリの利用状況（起動回数、画面遷移など）
- クラッシュ情報、エラーログ

これらの情報は、Google Firebase（Google Analytics for Firebase / Firebase Crashlytics）を利用して収集されます。

## 収集しない情報
本アプリでは、以下の情報を収集・外部送信することはありません。

- 氏名、メールアドレス等の個人を特定できる情報
- アプリ内で入力された曲名、アーティスト名、メモ等のユーザー入力データ

これらのデータは、すべてユーザーの端末内にのみ保存されます。

## 第三者提供について
収集した情報を、第三者に提供することはありません。
